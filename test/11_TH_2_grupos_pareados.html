9.1 Teste T pareado

  • Suposição de Normalidade das Diferenças. Nos dados pareados, a normalidade deve ser verificada nas diferenças e não nos valores de cada grupo.

Exemplo: Um pesquisador deseja verificar se há diferença entre os níveis de expressão do gene X antes e depois de um tratamento em uma cultura de células. Foram selecionados 12 amostras e a expressão do mRNA foi avaliada nos 2 momentos.

Hipóteses:

  • \(H_0\): Não há diferença na média da expressão gênica antes e depois do tratamento.

  • \(H_1\): Há diferença na média da expressão gênica antes e depois do tratamento.

Code
## Dados
Antes <- c(7.9, 8.5, 12.1, 9.1, 9.3, 12.4, 9.9, 6.5, 7.5, 8.1, 11.4, 9.7)
Depois <- c(10.7, 10.8, 13.0, 14.7, 12.3, 10.5, 13.3, 7.5, 7.5, 9.7, 11.3, 10.3)

dados <- data.frame(Antes, Depois)

# CALCULAR AS DIFERENÇAS
dados$D = dados$Depois - dados$Antes

Visualização dos dados

ggplot(dados, aes(y = D)) +
  geom_boxplot(fill = "lightblue", color = "darkblue", outlier.color = "red", outlier.shape = 16, width = 0.3) +

Teste de Normalidade

Code
# TESTE DE NORMALIDADE
shapiro.test(dados$D)

    Shapiro-Wilk normality test

Como p-valor igual a 0,9872 (>\(\alpha\) = 0,05), não rejeitamos a hipótese de normalidade dos dados.

Teste T pareado

t.test(Antes,

       conf.level=.95, 
       paired=TRUE,
       data = dados) #dados pareados

dados_long <- dados %>%
  select(Antes, Depois) %>% 
  mutate(ID = row_number()) %>%

dados_long %>%

9.2 Teste de Wilcoxon

  • Os dados não necessitam ser normais, mas devem ser no mínimo ordinais;

  • Há pouca diferença entre os Teste T e de Wilcoxon para distribuições próximas da normal.

Code
# teste de Wilcoxon
wilcox.test(dados$Antes, 
            dados$Depois, 
            alternative="two.sided", 
            paired=TRUE,
            correct=TRUE)

## Teste T pareado

-   Suposição de Normalidade das Diferenças. Nos dados pareados, a normalidade deve ser verificada nas diferenças e não nos valores de cada grupo.

**Exemplo**: Um pesquisador deseja verificar se há diferença entre os níveis de expressão do gene X antes e depois de um tratamento em uma cultura de células. Foram selecionados 12 amostras e a expressão do mRNA foi avaliada nos 2 momentos.

-   $H_0$: Não há diferença na média da expressão gênica antes e depois do tratamento.

Antes <- c(7.9, 8.5, 12.1, 9.1, 9.3, 12.4, 9.9, 6.5, 7.5, 8.1, 11.4, 9.7)
